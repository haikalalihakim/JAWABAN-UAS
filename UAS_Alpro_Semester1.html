1. Perbedan compiler, kode sumber akan dikonversi menjadi machine code sebelum program tersebut dijalankan. Sementara, Perbadaan interpreter mengkonversi source code menjadi machine code secara langsung ketika program dijalankan.
  2. perbedaan prosedur dan fungsi ada di pengembalian nilai, nilai hanya diproses didalam prosedur dan kembali kepada keadaan semula ketika keluar dari prosedur,Sedangkan pada fungsi, nilai yang diproses akan berubah dan memiliki nilai keluaran.
  3. Perbedaan Class yaitu, yang digunakan untuk mempelajari dan mereferensikan properti dari hierarki kelas berorientasi objek,Object yaitu, yang digunakan untuk memeriksa objek yang dipakai dalam program aplikasi yang sedang berjalan.
  4. Perbedaan Array dan ArrayList:
    *Array adalah struktur data di memori yang terdiri dari sekumpulan elemen yang memiliki tipe data yang sama dan setiap elemen data di dalam larik memiliki indeks sebagai acuannya
    *ArrayList merupakan array yang dapat diubah ukurannya, yang dapat ditemukan dalam paket java.util.
